# Wish List — Список желаний

Консольное приложение на Node.js для управления персональным списком желаний и целей.

## Функциональность

- Добавление желаний с названием, описанием, ссылкой на фото и приоритетом
- Приоритеты: «Срочно» и «Когда-нибудь»
- Отметка желания как выполненного с возможностью возврата
- Удаление желаний
- Фильтрация по статусу и приоритету
- Случайные мотивационные цитаты (20 штук)
- Сохранение данных в файл wishes.json

## Запуск

```bash
node src/app.js
```

## Технологии

- JavaScript ES6+
- Node.js (модули fs, readline)
