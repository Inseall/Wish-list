# Wish List — Список желаний

Веб-приложение для ведения личного списка желаний и целей.
Позволяет добавлять желания, устанавливать приоритеты, отслеживать прогресс и получать мотивационные цитаты.

## Функциональность

- Добавление желаний с названием, описанием и фото
- Приоритеты: «Срочно» и «Когда-нибудь»
- Фильтрация по статусу и приоритету
- Поиск по названию и описанию
- Отметка желания как выполненного
- Случайные мотивационные цитаты
